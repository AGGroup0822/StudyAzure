Notes on this sheet
  • Each day card shows a topic, a hands-on lab you can do in the free Azure sandbox or free account, a real-world use case for context, and a link to the official Microsoft Learn content.
  • AZ-305 labs are design exercises (diagrams, decision tables) rather than portal clicks — that's intentional, since AZ-305 tests architecture decisions, not implementation steps.
  • AZ-305 doesn't have per-domain Learn paths like AZ-104 does, so those days link to the official AZ-305T00 course outline or the Azure Architecture Center.
  • Progress is saved automatically in this browser — click the checkbox to mark a day complete.
  • Set a start date above to see a "TODAY" marker showing which day you should be on if you're on pace.
General resources
Sandbox labs
Microsoft Learn (free sandbox)

Every training module includes a free, time-limited Azure sandbox for hands-on exercises — no credit card needed.

Free subscription
Azure Free Account

$200 credit plus 12 months of select free services — needed once sandbox limits aren't enough for a lab.

Video deep dives
John Savill's Technical Training

Free YouTube channel widely regarded as the best deep-dive explainer for AZ-104/AZ-305 concepts.

Architecture reference
Azure Architecture Center

Reference architectures that AZ-305 case-study questions are built around — essential for the final phase.

Design framework
Azure Well-Architected Framework

The five pillars (reliability, security, cost, operations, performance) that underpin most AZ-305 case studies.

Enterprise adoption
Cloud Adoption Framework

Microsoft's methodology for landing zones and governance at scale — relevant to AZ-305's governance domain.

Sample templates
Azure Quickstart Templates (GitHub)

Hundreds of ready-made ARM/Bicep templates — useful for compute, networking, and Bicep-focused lab days.

Community Q&A
Microsoft Q&A — Azure tag

Official forum for asking questions when a lab doesn't behave as expected.

Stay current
Azure Updates

Official feed of new features and GA announcements — useful in the final AZ-305 review weeks.

Practice exams

Free practice tests co-developed with MeasureUp, matched to the real exam's style and difficulty.
