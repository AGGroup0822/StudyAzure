Notes on this sheet
  • Each day card shows a topic, a hands-on lab you can do in the free Azure sandbox or free account, a real-world use case for context, and a link to the official Microsoft Learn content.
  • AZ-305 labs are design exercises (diagrams, decision tables) rather than portal clicks — that's intentional, since AZ-305 tests architecture decisions, not implementation steps.
  • AZ-305 doesn't have per-domain Learn paths like AZ-104 does, so those days link to the official AZ-305T00 course outline or the Azure Architecture Center.
  • Progress is saved automatically in this browser — click the checkbox to mark a day complete.
  • Set a start date above to see a "TODAY" marker showing which day you should be on if you're on pace.
